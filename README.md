\# Emotion Detection with CNN



This project trains a CNN on the FER2013 dataset to detect emotions in real-time using a webcam.



\## Features

\- Train on FER2013 dataset (downloaded from Kaggle)

\- Save model in both `.h5` and `.tflite` formats

\- Real-time webcam emotion detection with OpenCV

\- Ready for mobile deployment via TensorFlow Lite



\## Usage

```bash

python emo\_detection.py
